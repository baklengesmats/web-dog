# web-dog

## Project setup
Requires Node.js version 8.9 or above (v10+ recommended). To build
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Standard local url running on http://localhost:8080/
### Compiles and minifies for production
```
npm run build
```
